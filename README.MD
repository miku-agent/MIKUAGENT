## 🌈 How to Use 🌈

1. Clone the repo! 🐾
    ```bash
    git clone https://github.com/agentdiva/AGENTDIVA.git
    ```
2. Navigate to the project directory! 🗂️
    ```bash
    cd AGENTDIVA
    ```
3. Install dependencies! 🎨✨
    ```bash
    npm install
    ```
4. Run the magic! 🎩🌟
    ```bash
    npm start
    ```
